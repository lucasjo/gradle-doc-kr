Working with existing builds

Installing Guide
----------------

#### 선제 조건

-	자바 JDK or JRE 7 이상<code>> java -version</code> Gradle는 자체 Groovy를 사용하고 있으므로 따로 설치 할 필요 없음. 기존 설치된 Groovy는 무시 됩니다. Gradle는 환경변수 JAVA_HOME이 가리키는 JDK를 인식하게 됩니다.
